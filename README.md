# Personal Homepage

This is a simple static personal website designed for GitHub Pages.

## Files

- `index.html`: page content
- `style.css`: page styling
- `assets/`: placeholder profile, publication, and logo images

## Deploy

Create a GitHub repository named:

```text
yourgithubusername.github.io
```

Replace `yourgithubusername` with your actual GitHub username, upload these files to the repository root, then enable GitHub Pages from the repository settings if needed.
